# Sistem Inventaris Barang (PHP)

Aplikasi berbasis web untuk mengelola stok barang masuk dan keluar di gudang.

## Fitur
- Dashboard Statistik Stok
- Kelola Data Barang, Supplier, dan Pelanggan
- Transaksi Barang Masuk & Keluar
- Laporan PDF

## Teknologi
- Backend: PHP Native (Tanpa Framework)
- Frontend: Bootstrap 5
- Database: MySQL

## Cara Instalasi
1. Import file `db_inventaris.sql` ke PHPMyAdmin.
2. Copy folder ini ke `htdocs`.
3. Buka browser: `localhost/sistem-inventaris-php`.
4. Login: admin / admin.# sistem-inventaris-php
